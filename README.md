# 🎬 K-Indie Film BEP Calculator

A cinematic and professional **Streamlit dashboard** designed to help estimate the **break-even point (BEP)** for Korean independent films.
This tool calculates how many admissions are needed to recover production costs based on ticket price and revenue distribution structure.

Perfect for:

* Independent filmmakers
* Film producers
* Film students
* Investors analyzing low-budget film economics

---

## ✨ Features

* 🎥 Film information input

  * Production budget
  * Marketing cost
  * Distribution fee
  * Other expenses

* 💰 Revenue distribution simulator

  * Theater share
  * Distributor share
  * Producer/Investor share

* 📊 Automatic calculations

  * Total production cost
  * Revenue per ticket
  * Break-even admissions
  * Required gross box office revenue

* 🎯 Audience scenario simulation

  * Pessimistic
  * Realistic
  * Optimistic

* 📈 Interactive visualizations with Plotly

  * Profit/Loss bar charts
  * Break-even comparison charts
  * Audience vs Profit line graph

* 🎞️ Fictional sample data for Korean indie films

## 🚀 How to Run

### 1. Clone the repository

```bash
git clone https://github.com/your-username/k-indie-film-bep-calculator.git
cd k-indie-film-bep-calculator
```

### 2. Install dependencies

```bash
pip install streamlit pandas plotly
```

### 3. Run the Streamlit app

```bash
streamlit run app.py
```

## 🎬 Example Use Case

An indie filmmaker can input:

* Production Budget: ₩80,000,000
* Marketing Cost: ₩15,000,000
* Average Ticket Price: ₩11,000

The dashboard then estimates:

* Required audience admissions
* Expected gross revenue
* Profit/loss across different audience scenarios
